In Informatics Group (IG), my major duty was problem solving. There were a lof of services that the group operates and therefore, there were a lot of problems. Here are the details of the work that I have done in IG.

  • Survey Service: I integrated the service with central LDAP service, making the service use the SSO mechanism. I also developed an approval system for Ethical Committee. I developed a very simple METU theme, and developed a secure and automated environment for users. These were for LimeSurvey 1.9x series. I was working for LimeSurvey 2.x upgrade but I wasn't able to finish it due to time limitations.
  • Drupal Work: IG uses Drupal as a framework for most of the applications they used. I integrated several external applications as Drupal plugins. I implemented Drush in all Drupal installations (single or multi-site). I was involved in porting existing theme to Drupal 7 native structure and did major work by doing heavy PHP coding in the template hooks. I migrated Drupal 6 installations to Drupal 7 having completelely different database structure, as well as transferring custom coded applications to Drupal structure. In conclusion, after 2 years of work, I can do anything with Drupal and Drush. Just ask for it :)
  • Websites in METU: IG provides support for personal, departmental and administrative unit websites. This support includes PHP coding, HTML-CSS-JS support, as well as Drupal work. Personal websites can only run HTML and uses tilde (~) in their addresses. I did an experimental application to remove tilde from the addresses, while web server still reads the user account's files (i.e. disabled mod_userdir and implemented a new method using mod_rewrite). Due to management issues, this system was never implemented but at the time of this writing, experimental system was still working. You can check it from here and here.
  • Blog Service: I received a slow working, an obsolete multi-site Wordpress installation at first. I managed to make that work and respond faster. I added several options to the service, developed user statistics and analysis plugins (mainly for inter-group purposes). I upgraded the service to the latest possible version and implemented single-click update and theme / plugin installation system. I was also involved in development of METU themes for Wordpress and did major work in custom coding, according to the requirements.
  • Electronic List Service: In general, I was involved in management of this service. I analyzed the problems, did some server-side security operations and that's all. I know the internals of Mailman and I like it very much.
  • Software Development: I worked for solving problems in CAS and BBS service, and solved several of them but due to management issues, they never got implemented. I also worked in development of Press Archive Service. I did a lot of small to large development, such as analysis, statistics and automation systems. Many of them are still in use, but I couldn't find time to deploy several additional services.
  • Server Management: I had no root access but I tested all the software in my local virtual machines, and then tell the server admins to install defined packages or compile additional software. I generally sat behind server admins to observe what was going on and when encountered, involve in problems related to my work.
  • Additionally, I did a lot of assistance in software development, planning and server management topics which are related or affecting my core work.
